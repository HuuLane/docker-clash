# docker-clash

https://hub.docker.com/r/junxy/clash

```bash
docker pull junxy/clash
```

- docker image 内置了 Clash UI 静态文件
- clash 最新 dev 分支

使用说明：

修改 `config/config.yaml` （参考 https://github.com/Dreamacro/clash#toc4 ），然后运行 `docker-compose up -d`
